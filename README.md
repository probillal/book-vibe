📚 Book Vibe

Book Vibe is a modern web application where users can explore books, view details, and manage their reading preferences. It provides a smooth and interactive UI to enhance the overall reading experience.

🚀 Live Demo
🔗 https://book-resource-project.netlify.app/

📂 GitHub Repository
🔗 https://github.com/probillal/book-vibe

✨ Features

📖 Browse and explore books

🔍 View detailed information about each book,
❤️ Mark books as read or wishlist,
⚡ Fast and responsive UI,
📱 Mobile-friendly design,

🛠️ Tech Stack

React

Tailwind CSS

JavaScript (ES6+)

React Router

State Management

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/probillal/book-vibe.git

# Navigate to project folder
cd book-vibe

# Install dependencies
npm install

# Run the project
npm run dev
